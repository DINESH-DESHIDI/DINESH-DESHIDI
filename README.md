<details>
<summary><b>🎓 College Discovery Platform</b></summary>

### College Discovery Platform
An interactive full-stack platform transforming raw TS EAMCET cutoff PDFs and government counseling data into real-time, actionable insights for engineering aspirants.

#### 📊 Architecture & Specifications
| Metric | Details |
| :--- | :--- |
| **Stack** | React.js, Node.js, Express.js, PostgreSQL, Prisma ORM, Neon DB, JWT, Tailwind CSS |
| **Scale** | Serves thousands of aspirants analyzing thousands of cutoff combinations and seat configurations. |
| **Performance** | Optimized query design with Prisma ORM indexing, yielding sub-100ms response times for complex filter queries. |
| **Security** | Secure authentication and session management via JSON Web Tokens (JWT) and HTTPS. |
| **Impact** | Eliminated manual page-by-page PDF analysis, cutting college search and planning time by over 90%. |
| **Repository** | [GitHub Repo](https://github.com/DINESH-DESHIDI/college-discovery-platform) |

#### 💡 Project Explanation
The TS EAMCET counseling process involves matching ranks across multiple branches, student categories, and college cutoffs. By structuring database tables for colleges, branches, and seats, this application lets users search, compare, and run "eligibility simulations" based on their category and rank.
</details>

An advanced AI operating companion combining voice interaction, autonomous reasoning, desktop automation, software development, memory management, and computer vision.

#### 📊 Architecture & Specifications

#### 💡 Project Explanation
JARVIS-MT67 merges advanced multimodal AI models with local operating system hooks. It can capture screen content via OpenCV, translate voice inputs via Whisper, and use Playwright and PyAutoGUI to navigate and execute software actions on the user's behalf.
